# unidad8 - Curso Python 3.8 UTN

Ejercicio N°8 con agregados propios para su experimentación.

Interfaz gráfica, con la que se puede crear una BD en MySQL llamada "mi_plantilla2"
Tambien crea una tabla en caso de que no exista llamada "productos"

Se pueden agregar 3 valores a la BD. 
  - Titulo
  - Ruta
  - Descripción
  
 Se agregan los valores a un Treeview.
 
 Luego se pueden Modificar o Borrar los valores de la BD desde sus respectivos botones en la interfaz.
